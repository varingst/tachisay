# tachisay

This is a very silly thing.

`tachisay` prints an ascii-art [tachikoma](https://en.wikipedia.org/wiki/Tachikoma) and a speech bubble.

Like [cowsay](https://en.wikipedia.org/wiki/Cowsay), only written in awk, not perl.

`tachigreeter.start` is for putting in `/etc/local.d/` if you're running
OpenRC. It creates an /etc/issue for your login screen.
